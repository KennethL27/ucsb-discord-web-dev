Bot Guide

Broida

Broida is one of main Discord Bots that we have developed and deployed within the server. The main purpose behind this project is to provide useful information to both moderators and users. Broida is made by using Discord's Python API. As the UCSB Physics Discord Community continues to grow, Broida will become more intricate and provide more usufull tools. To view its full list of commands that you can use simply send a ".help" (preferably in the bot command channel) which will provide you a list of the commands that Broida has to offer.
